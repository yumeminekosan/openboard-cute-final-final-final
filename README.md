# 猫咪输入法 ฅ^•ﻌ•^ฅ

一个简单的Android软键盘，带有猫咪文字变身功能！

## 功能

- 基本QWERTY键盘布局
- 中文输入猫咪变身
- 猫咪颜文字快捷键

## 猫咪变身规则

| 原文 | 变身后 |
|-----|--------|
| 我 | 猫 |
| 你/他/她/它 | 人咪 |
| 人 | 人咪 |
| 个 | 只 |
| 句末标点 | +喵~ |

**示例：**
- 「我是一个人。」→ 「猫是一只人咪喵~。」
- 「你好！」→ 「人咪好喵~！」

## 构建

```bash
./gradlew assembleDebug
```

APK位置：`app/build/outputs/apk/debug/app-debug.apk`
